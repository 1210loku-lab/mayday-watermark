# 回到那一天 · 纪念水印工具

五月天 2026 北京鸟巢「#5525+2 回到那一天·无限延伸版」演唱会粉丝专属纪念水印工具。

**在线使用：** [mayday-watermark.top](https://www.mayday-watermark.top/)

---

## 功能

- 上传演唱会现场照片，一键叠加主题水印
- 固定水印：左上角 MAYDAY 品牌标识 + 右上角演出日期与北京·鸟巢
- 四种可选水印风格：
  - **五球狂欢** — 五位成员专属彩色球排列（蓝·冠佑 / 黄·玛莎 / 粉·阿信 / 红·怪兽 / 绿·石头）
  - **五球合体** — 五球合体纪念图
  - **MOJO Family** — MOJO Family 联名水印
  - **歌词模式** — 自定义歌词文字水印
- 自定义留言、演出场次、字体、字号、透明度及位置偏移
- 所有照片**本地处理**，不上传至任何服务器
- 输出照片与原图分辨率一致，无质量损失
